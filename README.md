# 3d-blog 项目

# 运行项目

1. 安装依赖

```bash
npm install
```

2. 运行项目

```bash
npm run dev
```

# 项目说明

这是一个基于 React 和 Vite 搭建的 3D 博客项目，使用了 Three.js 和 React-Three-Fiber 来实现 3D 效果。
项目包含了地面、灯光、摄像机控制以及一个白色石雕模型的展示。
项目结构清晰，易于扩展和维护，适合用于学习和展示 3D 技术在 Web 开发中的应用。

# 主要功能

-   地面和环境设置
-   灯光和阴影效果
-   石雕模型展示
-   人物运动控制
-   艺术画展示
-   环境贴图展示
-   反射材质效果
